## Run Locally

Clone the project

```bash
  git clone https://github.com/allanmonth/weather
```

Go to the project directory

```bash
  cd weather
```

Install dependencies

```bash
  yarn i
```

Start the server

```bash
  yarn dev
```

Start test

```bash
  yarn test
```
